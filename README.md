
# Advanced Portfolio – Abdurhim Ali

Modern professional portfolio website with:

- Modern landing page
- Dark mode
- Skills chart
- Project gallery
- Resume download
- Animated sections ready

## Deploy with GitHub Pages

1. Create repository `abdurhim-portfolio`
2. Upload all files
3. Go to Settings → Pages
4. Enable `Deploy from main branch`

Your website will be live at:

https://yourusername.github.io/abdurhim-portfolio
